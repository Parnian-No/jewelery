# پری گلد - Pari Gold

وب‌سایت و کاتالوگ برند پری گلد  
طراح: پرنیان نوروزی

## درباره پروژه

این پروژه شامل یک وب‌سایت زیبا با تم طلایی و مشکی و یک کاتالوگ ۲۰ صفحه‌ای از جواهرات الهام‌گرفته از حشرات است.

## ویژگی‌ها

- ✅ طراحی مدرن با تم طلایی و مشکی
- ✅ واکنش‌گرا (Responsive Design)
- ✅ کاتالوگ ۲۰ صفحه‌ای کامل
- ✅ قابلیت دانلود PDF از کاتالوگ
- ✅ رابط کاربری فارسی (RTL)

## فایل‌های پروژه

- `index.html` - صفحه اصلی وب‌سایت
- `styles.css` - استایل‌های صفحه اصلی
- `catalog.html` - صفحه کاتالوگ ۲۰ صفحه‌ای
- `catalog.css` - استایل‌های کاتالوگ
- `catalog.js` - اسکریپت تولید PDF

## نحوه استفاده

### مشاهده محلی

1. تمام فایل‌ها را در یک پوشه قرار دهید
2. فایل `index.html` را در مرورگر باز کنید
3. برای مشاهده کاتالوگ، روی دکمه "مشاهده کاتالوگ PDF" کلیک کنید

### تولید PDF

1. صفحه `catalog.html` را باز کنید
2. روی دکمه "دانلود PDF" کلیک کنید
3. صبر کنید تا PDF تولید شود (ممکن است چند لحظه طول بکشد)

### انتشار در GitHub Pages

1. این پروژه را در GitHub آپلود کنید
2. به Settings > Pages بروید
3. منبع (Source) را روی "main branch" تنظیم کنید
4. بعد از چند دقیقه، سایت شما در آدرس زیر در دسترس خواهد بود:
   ```
   https://[username].github.io/[repository-name]/
   ```
